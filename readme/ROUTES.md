# Routes Documentation

## Цель документа

Этот документ фиксирует маршруты MVP веб-сайта MTG Global Stats. Агент должен использовать эти роуты при настройке React Router.

## Public routes

Все публичные страницы доступны без авторизации.

### `/`

Главная страница статистики.

MVP default filters:

```text
cityId = moscow
formatId = legacy
clubId = not passed
```

Показывает:

- summary cards;
- последние турниры;
- метагейм колод;
- performance колод;
- топ игроков;
- популярные матчапы.

API:

```http
GET /api/v1/home?cityId=moscow&formatId=legacy
```

### `/tournaments`

Список турниров.

Показывает турниры с фильтрами:

- город;
- клуб;
- формат;
- тип турнира;
- период.

API:

```http
GET /api/v1/tournaments
```

### `/tournaments/:id`

Детальная страница турнира.

Показывает:

- header турнира;
- финальные стендинги;
- все раунды/паринги;
- список игрок -> колода;
- метагейм турнира, если доступен.

API:

```http
GET /api/v1/tournaments/:id
```

### `/players`

Список игроков.

Показывает игроков и их краткую статистику по выбранным фильтрам.

API:

```http
GET /api/v1/players
```

### `/players/:id`

Детальная страница игрока.

Показывает:

- summary игрока;
- турниры игрока;
- колоды игрока;
- историю матчей, если будет добавлена позже.

API:

```http
GET /api/v1/players/:id
```

### `/decks`

Список колод.

Показывает колоды и их краткую статистику по выбранным фильтрам.

API:

```http
GET /api/v1/decks
```

### `/decks/:id`

Детальная страница колоды.

Показывает:

- summary колоды;
- результаты колоды по турнирам;
- игроков, которые играли этой колодой;
- матчапы, если будут добавлены позже.

API:

```http
GET /api/v1/decks/:id
```

## Admin routes

На MVP авторизация может быть не реализована, но route для создания турнира должен быть отделен от публичных страниц.

### `/admin/tournaments/create`

Экран создания турнира вручную.

Поля:

- дата;
- город;
- клуб;
- тип турнира;
- формат;
- CSV файл финальных стендингов;
- CSV файл всех раундов;
- текстовый список игрок -> колода.

API:

```http
GET /api/v1/cities
GET /api/v1/cities/:cityId/clubs
GET /api/v1/formats
POST /api/v1/admin/tournaments/import
```

## Navigation

Основная навигация:

```text
Главная
Турниры
Игроки
Колоды
Добавить турнир
```

`Добавить турнир` может быть видимой ссылкой в MVP, но в будущем она должна быть доступна только администраторам.

## 404

Для неизвестных маршрутов сделать страницу:

```text
Страница не найдена
```

С кнопкой перехода на главную.

## Redirects

Если пользователь открывает `/home`, можно редиректить на `/`.

## Future routes
